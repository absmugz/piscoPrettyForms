Una forma simple y sencilla de añadir tus diseños personalizados a los elementos de formularios que no permiten CSS, como selects, inputs radios y checkboxes.

Intro

Pisco pretty Forms permite añadir tus propios diseños a los selects de un formulario así como a las casillas de verificación (inputs radios y checkboxes)

No es el único que existe, pero si está a la par con el más ligero y sencillo de usar. Es simplemente una elección más.

Uso para 'selects'

El uso básico es muy simple, añade la clase que dispara el plugin a tu select y llámalo:

<select class="prettyForm" data-title="Elige tu deporte favorito">
   <option value="1">Ciclismo</option>
   <option value="2">Fútbol</option>
   <option value="3">Natación</option>
   <option value="4">Running</option>
   <option value="5">Baloncesto</option>
</select>

<select class="prettyForm">
   <option data-first="title">Con qué frecuencia lo practicas?</option>
   <option value="1">1 vez al mes</option>
   <option value="2">De 2 a 5 veces almes</option>
   <option value="3">Más de dos veces a la semana</option>
   <option value="4">Todos los días</option>
</select>

<script>
$('.prettyForm').piscoPrettyForms();
</script>

Por defecto busca en el select el atributo data-title para ponerlo como titulo del mismo, pero si este no está, se usará el primer option. Para no duplicar información, el sscript eliminará el option que tenga el atributo data-first. Se trata de dos forms de hacer lo mismo para elegir la que mejor se adapte a tu proyecto.

Opciones para 'selects'

Por defecto el tamaño del selector es automático, adaptado al tamaño de la cadena de texto del título más el relleno y la flechita. No obstante se puede poner un tamaño fijo:

<script>
$('.prettyForm-fixedSize').piscoPrettyForms({
	selectWidth: 350
})
</script>

O si necesitas más control, y diseños personalizados, puedes añadirle una clase personalizada y darle estilos por CSS:

$('.prettyForm-customClass').piscoPrettyForms({
	customClass: 'custom-class'
});</script>

Uso para 'radio buttons'

El uso básico es muy simple, añade la clase que dispara el plugin a tu input radio y llámalo. Los estilos del mismo puedes modificarlos directamente por CSS:

<script>
$('.prettyForm').piscoPrettyForms();
</script>





Y también puedes añadir una clase personalizada por si lo necesitas:

<script>
$('.prettyForm-radioCustom').piscoPrettyForms({
   customClass: 'radioCustom'
});
</script>



Uso para 'checkboxes (casillas de verificación'

Al igual que en el caso de lso radio buttons, añade la clase que dispara el plugin a tu input radio y llámalo. Los estilos del mismo puedes modificarlos directamente por CSS:

<script>
$('.prettyForm').piscoPrettyForms();
</script>
